F L A M E S is a game named after the acronym: Friendship, Love, Affection, Marriage, Enemity, Sibblings. It is a popular game among young girls at sleepovers, especially with pre-teens and teens who are beginning to explore the world of crushes. This game does not accurately predict whether or not an individual is right for you, but it can be fun to play this with your friends.

It is a fun filled love calculator to test the status of your love relationship between him/her using FLAMES. Asimple test of love using only names of people.

Algorithm

It is very easy to explain with an example:
Your name: asd
Partner name: abcd
There are two main steps:

  1. Get the flames count
    • Consider two names ('asd' and 'abcd')
    • Remove common characters (two common characters 'a', 'd')
    • Count the number of characters left (Removed a,d and the rest are s,b,c. So total 3.)
  2. Determine the flames result
    • We take FLAMES letters ('F', 'L', 'A', 'M', 'E', 'S')
    • And start removing letters using the flames count we got.
    • And the letter which last the process is the result.

In our example we got flames count = 3. So first we take the letters FLAMES.
F L A M E S
We, then start counting from left till the flames count 3. Then remove the letter which is at position 3. In this case it is 'A'. So the letters become:
F L M E S
Then, we start counting again from the letter which was removed i.e., from 'M'. So the next character to remove is 'S'. So our letters become:
F L M E
After next step:
F L E
Then:
F E
Last:
F
So the result is 'Friendship'.
